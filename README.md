# Mental Health Self-Care Tracker

Group project for CS 4398/5394 Software Engineering.

This JavaFX desktop application helps users track moods, habits, and personal well-being.

## Tech Stack
- Java (JDK 17)
- JavaFX
- Maven
- SQLite
- MVC Architecture

## My Contributions
- Login & session management
- Dashboard navigation (Profile, Habits, Mood Tracker)
- Mood tracking + history feature
- UI integration using JavaFX (FXML controllers)
- Iteration reports and documentation

## How to Run
mvn clean javafx:run

## Note
This was a group project developed with teammates. This repository is a personal copy for portfolio purposes.
